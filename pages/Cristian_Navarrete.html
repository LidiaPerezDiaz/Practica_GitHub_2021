UF2 - Comandos Git

git config --global user.

El comando git config --global user. es utilizado para definir el usuario global en el equipo que va a trabajar en git, el orden sería introducir git config --global user.email xxxxxxxx@xxxx.xxx seguido de git config --global user.password xxxxx Verificamos que esté correcto haciendo git config --list y si aparece nuestro nombre y correo, lo tenemos listo. Una vez hemos hecho esto, habremos definido el usuario que va a trabajar en git y estaremos listos para subir los cambios que hagamos al contenido
